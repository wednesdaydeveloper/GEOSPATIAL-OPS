# GEOSPATIAL OPS

**管制室／ハッカー風のフルスクリーン・ダッシュボード。** 中心では実データの海岸線・大陸で描いた地球儀が自転し、人工衛星が周回し、各種テレメトリパネルや暗号解読風の演出がリアルタイムに動きます。

> いかにも高度な作業をしているように「見える」演出用ダッシュボードです。実際のネットワーク通信や解読は一切行わず、すべて見た目だけの演出です。

🛰️ **ライブデモ: https://wednesdaydeveloper.github.io/GEOSPATIAL-OPS/**

（全画面表示が効果的です。Mac は `⌃⌘F`）

---

## 特徴

- **依存ライブラリ・ビルド・通信は一切不要。** `index.html` をブラウザで開くだけで動作します。
- **完全オフライン動作。** 海岸線データと陸地マスクはファイルに埋め込み済み（Base64 等）。
- **単一ファイル構成。** すべてが `index.html` 一枚に収まっています。

## 主な演出

### 地球儀
- **自転する地球** — Y 軸（極軸）まわりに回転。
- **大陸の塗り** — 陸地ビットマップを球面に逆投影してピクセル単位で描画。
- **海岸線・経線・赤道・地軸** — 前面は明るく、背面は薄く描き分け。
- **大阪マーカー** — 脈動リング＋十字＋ラベル（前面のときだけ表示）。

### 人工衛星
- 6 基の衛星がそれぞれ異なる軌道半径・傾斜角・位相・速度で周回。
- 本体＋ソーラーパネルのグリフを動径方向に姿勢合わせして描画。
- 軌道リングとトレイルつき。

### ハッカー風エフェクト
- **コードレイン** — 地球の背後を流れる文字の雨。
- **16進パケットダンプ** — 傍受風の hex 表示。
- **デクリプト演出** — 文字スクランブルが左から確定していく解読アニメ。
- **シグナル再構築球** — 緯度経度クアッドが「穴だらけ → 完成」へ復元されるループ。
- **グリッチタイトル / ASCII アート** — RGB ずれのタイトルと衛星のアスキー絵。

### UI
- 上バー / 左カラム（SIGINT・テレメトリ・イベントログ）/ 中央ステージ / 右カラム（衛星ロスター・再構築球・スペクトラム）/ 下ティッカー。
- 配色は CSS の `:root` カラートークンで一元管理。

## 操作

- **横ドラッグ** で地球と軌道を回転（縦ドラッグは無効）。
- ドラッグ中は自動回転が停止し、手を離して約 2.5 秒後に再開。

## 動かし方

ローカルで開く場合:

```sh
git clone https://github.com/wednesdaydeveloper/GEOSPATIAL-OPS.git
cd GEOSPATIAL-OPS
open index.html        # macOS（Windows なら start index.html）
```

`index.html` をダブルクリックするだけでも動きます。ビルドやサーバーは不要です。

## ファイル構成

単一ファイル `index.html` のみ。中身は大きく 4 ブロック:

1. `<style>` — CSS。配色は `:root` のカラートークンで管理。
2. データ用 `<script>` — `COAST`（陸地リングの座標）を埋め込み。
3. メイン `<script>` 冒頭 — `MASK_B64`（陸地ビットマップ）を埋め込み。
4. メイン `<script>` 残り — 描画・アニメーションの全ロジック。

> 開発・編集時の詳細（座標系と投影の約束事、各モジュールの調整ポイント、既知の落とし穴、埋め込みデータの再生成手順）は [CLAUDE.md](./CLAUDE.md) を参照してください。

## 技術的なポイント

- **正射投影で統一** — 塗り（マスク逆投影）と線がズレないよう、透視投影は使わず正射に統一。
- **東西反転対策** — 投影の画面 X は `CX - X`、塗りの逆投影は `X = -nx` をセットで適用。
- **マスク塗り方式** — 巨大リングでベクタ塗りが破綻するため、陸地ビットマップ（480×240）を球面に逆投影する方式を採用。
- **衛星の軌道半径** — リサイズ後の半径 `R` を使うため、軌道半径は必ず描画時に計算。

## データ出典

海岸線・陸地マスクは [Natural Earth](https://www.naturalearthdata.com/) の 110m データ（`ne_110m_land` 等）から生成し、ファイルに埋め込んでいます。

## ライセンス

このリポジトリのコードは自由に利用できます。地理データの利用については Natural Earth のパブリックドメイン方針に従います。
